一、环境要求

1. Environment Requirements

在安装和运行此VPN客户端&管理系统之前，请确保你的服务器满足以下环境要求：

Web 服务器

Apache 2.4 或 Nginx 1.14 及以上版本

PHP 版本

PHP 8.1 或更高版本

数据库

MySQL 5.6 或更高版本

腾讯云账号

腾讯云账号及相应的 API 密钥

Before installing and running this VPN Client & Management System, please ensure your server meets the following requirements:

Web Server

Apache 2.4 or Nginx 1.14 and above

PHP Version

PHP 8.1 or higher

Database

MySQL 5.6 or higher

Tencent Cloud Account

Tencent Cloud account and corresponding API keys

二、安装步骤

2. Installation Steps

1 下载项目代码

1 Download Project Code

从 GitHub 仓库克隆项目代码到你的服务器：

git clone <项目仓库地址>

Clone the project code from the GitHub repository to your server:

git clone <Project Repository URL>

2 配置数据库

2 Configure Database

打开 config.php 文件，配置数据库连接信息：

// 数据库配置
define('DB_HOST', 'localhost');
define('DB_NAME', 'your_database_name');
define('DB_USER', 'your_database_user');
define('DB_PASSWORD', 'your_database_password');

请将 your_database_name、your_database_user 和 your_database_password 替换为你自己的数据库名称、用户名和密码。

Open the config.php file and configure the database connection information:

// Database Configuration
define('DB_HOST', 'localhost');
define('DB_NAME', 'your_database_name');
define('DB_USER', 'your_database_user');
define('DB_PASSWORD', 'your_database_password');

Please replace your_database_name, your_database_user, and your_database_password with your own database name, username, and password.

3 配置腾讯云 API

3 Configure Tencent Cloud API

同样在 config.php 文件中，配置腾讯云 API 信息：

// 腾讯云API配置
define('SECRET_ID', 'your_secret_id');
define('SECRET_KEY', 'your_secret_key');
define('REGION', 'your_region');
define('INSTANCE_ID', 'your_instance_id'); // VPN服务器实例ID

请将 your_secret_id、your_secret_key、your_region 和 your_instance_id 替换为你自己的腾讯云 API 密钥、区域和 VPN 服务器实例 ID。

Also in the config.php file, configure Tencent Cloud API information:

// Tencent Cloud API Configuration
define('SECRET_ID', 'your_secret_id');
define('SECRET_KEY', 'your_secret_key');
define('REGION', 'your_region');
define('INSTANCE_ID', 'your_instance_id'); // VPN Server Instance ID

Please replace your_secret_id, your_secret_key, your_region, and your_instance_id with your own Tencent Cloud API keys, region, and VPN server instance ID.

4 创建数据库表

4 Create Database Tables

使用 MySQL 客户端连接到你的数据库，并执行以下 SQL 语句创建所需的表：

-- 创建用户表
CREATE TABLE users (
id INT AUTO_INCREMENT PRIMARY KEY,
username VARCHAR(255) NOT NULL UNIQUE,
password_hash VARCHAR(255) NOT NULL,
is_active TINYINT(1) DEFAULT 0,
created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 创建VPN连接记录表
CREATE TABLE vpn_connections (
id INT AUTO_INCREMENT PRIMARY KEY,
user_id INT NOT NULL,
connection_time DATETIME NOT NULL,
disconnection_time DATETIME,
ip_address VARCHAR(45) NOT NULL,
country VARCHAR(100),
bytes_sent BIGINT DEFAULT 0,
bytes_received BIGINT DEFAULT 0,
FOREIGN KEY (user_id) REFERENCES users(id)
);

-- 创建操作日志表
CREATE TABLE operation_logs (
id INT AUTO_INCREMENT PRIMARY KEY,
user_id INT NOT NULL,
operation TEXT NOT NULL,
type VARCHAR(255) NOT NULL,
created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Connect to your database using a MySQL client and execute the following SQL statements to create the required tables:

-- Create Users Table
CREATE TABLE users (
id INT AUTO_INCREMENT PRIMARY KEY,
username VARCHAR(255) NOT NULL UNIQUE,
password_hash VARCHAR(255) NOT NULL,

);

-- Create VPN Connections Table
CREATE TABLE vpn_connections (
id INT AUTO_INCREMENT PRIMARY KEY,

connection_time DATETIME NOT NULL,
disconnection_time DATETIME,
ip_address VARCHAR(45) NOT NULL,
country VARCHAR(100),
bytes_sent BIGINT DEFAULT 0,

-- Create Operation Logs Table
CREATE TABLE operation_logs (
id INT AUTO_INCREMENT PRIMARY KEY,
user_id INT NOT NULL,
operation TEXT NOT NULL,
type VARCHAR(255) NOT NULL,
created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

三、访问系统

3. Access the System

完成上述步骤后，你可以通过浏览器访问系统的登录页面：

http://your_server_domain/login.php

请将 your_server_domain 替换为你服务器的域名或 IP 地址。

After completing the above steps, you can access the system's login page via your browser:

http://your_server_domain/login.php

Please replace your_server_domain with your server's domain name or IP address.

四、注意事项

4. Notes

安全建议

请确保你的服务器安全，建议使用防火墙和 SSL 证书。

数据备份

定期备份数据库，以防止数据丢失。

故障排除

如果遇到问题，请查看服务器日志文件以获取更多信息。

Security

Please ensure your server is secure. It is recommended to use firewalls and SSL certificates.

Backup

Regularly back up your database to prevent data loss.

Troubleshooting

If you encounter any issues, please check the server log files for more information.
